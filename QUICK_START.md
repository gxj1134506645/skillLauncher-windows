# Skill Launcher Windows - 一键安装指南

## 🚀 3 秒完成安装

### 1️⃣ 下载安装

访问 GitHub Releases 页面，下载：
```
Skill Launcher_1.0.0_x64-setup.exe
```

### 2️⃣ 双击运行

双击下载的 `setup.exe`，点击"下一步"直到安装完成。

**✅ 安装程序会自动：**
- 安装应用到电脑
- 配置 Claude Code skill
- 创建桌面快捷方式

### 3️⃣ 立即使用

完全重启 Claude Code，然后输入：
```
/skill-launcher
```

**就这么简单！** ✨

---

## 📦 安装后你将获得

- ✅ Skill Launcher 应用（可独立运行）
- ✅ Claude Code CLI 集成（`/skill-launcher` 命令）
- ✅ 桌面快捷方式
- ✅ 开始菜单项
- ✅ 全局快捷键 `Ctrl+Alt+Space`

## 🎯 使用方法

### 启动方式

**方式 1：通过 Claude Code CLI（推荐）**
```
/skill-launcher
```

**方式 2：桌面快捷方式**
双击桌面的 "Skill Launcher" 图标

**方式 3：全局快捷键**
按 `Ctrl+Alt+Space`

## 🔧 功能特性

- **智能搜索** - 输入关键词快速查找 skills

- **智能排序** - 常用的 skills 自动排在前面

## ❓ 常见问题

### Q: 安装后输入 `/skill-launcher` 没反应？
**A:** 确保完全重启了 Claude Code CLI（关闭所有窗口后重新打开）

### Q: 想要卸载怎么办？
**A:** 通过"设置 → 应用"卸载，或使用卸载程序

### Q: 可以自定义快捷键吗？
**A:** 可以！在应用设置中修改快捷键
