# Skill Launcher for Windows

当用户输入 `/skill-launcher` 时，打开 Skill Launcher 桌面端窗口。

执行方式（固定）：
```
cmd /c launch.bat
```

注意：
- 应用窗口会展示项目级与用户级 skills，点击即可复制 skill 名称到剪贴板。
- 如果启动失败，请检查是否已安装 Skill Launcher 或 `launch.bat` 路径是否可用。
