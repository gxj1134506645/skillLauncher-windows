# SkillLauncher Windows

**Windows 版 Claude Code Skills 快速启动器**

[![Download](https://img.shields.io/badge/Download-Latest-blue)](https://github.com/gxj1134506645/skillLauncher-windows/releases/latest) ![Tauri](https://img.shields.io/badge/Tauri-2.0-orange)

基于 Tauri 2 + React 18 的 Windows 版 Claude Code Skills 启动器，键盘优先用于 CLI 内快速搜索与执行。

</div>

---

## 安装

- 下载 [Skill_Launcher_x64-setup.exe](https://github.com/gxj1134506645/skillLauncher-windows/releases/latest) 并运行
- 直接默认安装，千万不要修改安装路径

---

## 使用

- 在 Claude Code CLI 中输入 `/skill-launcher`

![image-20260204172234829](https://cdn.jsdelivr.net/gh/gxj1134506645/img-bed@main/images/20260204172234915.png)

---

## 技术栈

React 18 + TypeScript + Fluent UI + Tauri 2.0 + Vite + Rust

---

## 开发

```powershell
npm install
npm run tauri dev
npm run tauri build
```
